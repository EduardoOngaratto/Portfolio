Sobre mim

Olá! Meu nome é Eduardo Ongaratto e sou um entusiasta de tecnologia focado em desenvolvimento de software, especialmente no lado do servidor. Minha paixão pelo mundo do back-end começou quando descobri a flexibilidade e poder do Node.js.

Sou um estudante de Ciência da Computação na Universidade URI Erechim, com sólida experiência em linguagens como C, C++, JavaScript (incluindo Node.js), TypeScript e Java. Além disso, domino HTML, CSS e React para desenvolvimento web, e tenho conhecimento em bancos de dados relacionais (MySQL, SQL Server) e NoSQL (MongoDB). Sou versátil e capaz de enfrentar desafios de desenvolvimento de software de forma eficiente, seja criando APIs robustas com Node.js ou aplicativos escaláveis com Java. Estou entusiasmado para aplicar meus conhecimentos em projetos desafiadores e contribuir para o sucesso de sua equipe ou projeto. Vamos criar algo incrível juntos!
